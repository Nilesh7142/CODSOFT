## 💻 Project Overviews

### 1. To-Do List Application
A smart task management tool designed to help users track their daily objectives.
* **Features:** Add tasks, toggle completion states with visual indicators (`[✓]`), delete tasks, and automatically save lists to a local JSON file (`todo_list.json`) so your progress is never lost.
* **File Name:** `todo.py`

### 2. Simple Calculator
A sleek, modern digital calculator mimicking smartphone layouts.
* **Features:** Clean grid design, real-time expression rendering, math parsing supporting standard operator precedence (PEMDAS/BODMAS), clear data functions (`C`), and crash prevention for zero-division errors.
* **File Name:** `gui_calculator.py`

### 3. Password Generator
A security-focused utility that generates cryptographically strong random passwords.
* **Features:** Customizable length constraint slider (6 to 32 characters), selectable complexity tokens (Uppercase, Numbers, Symbols), and a single-click "Copy to Clipboard" implementation for immediate use.
* **File Name:** `password_gen.py`

## 🛠 Prerequisites

To run these applications, you need **Python 3.x** installed on your system. 

> ⚠️ **Important (Windows Users):** Ensure you check the **"Add Python to PATH"** checkbox during the Python installation process.

Since all applications utilize **Tkinter** (Python's built-in GUI toolkit) and standard modules (`json`, `random`, `string`), **no external pip installations are required!**
